# README 多智能体信息图设计说明

## 目标
- 在 README 中用一张图快速说明项目有哪些 agent、各自职责、接入了哪些第三方数据源。
- 让首次浏览仓库的人在 10 秒内理解项目的研究链路。

## 视觉方向
- 采用第二张图：手绘涂鸦漫画风。
- 让 agent 成为主角，tools 和 data sources 为配角。
- 整体保持轻松、不严肃，但信息结构清晰。

## 图中要表达的内容
- 左侧：第三方数据源与输入工具。
- 中间：6 个分析 agent 与 1 个最终汇总 agent。
- 右侧：预测结果和交易执行字段。
- 底部：历史反馈闭环。

## README 配套文本
- 开头简要介绍项目定位。
- 紧接着放信息图。
- 然后用表格列出 agent 职责和第三方数据源。
- 最后补充 workflow 概览、开发方式和免责声明。
